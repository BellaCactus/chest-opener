## what is this?

**mr mine chest opener** is a simple python script that helps automate the repetitive part of opening chests.

this is mainly:
- a tiny quality-of-life tool
- a record of a small automation script
- a “practical python” mini project

---

## features

- simple automation flow (low effort, quick runs)
- intended for repetitive chest opening
- minimal setup (depends on how the script is written)
